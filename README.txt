1. Please make sure you have Ollama installed
You can do this from https://ollama.com/download
2. Please pull llama3 in your command prompt using 'ollama pull llama3' in your environment prompt
3. Please make sure you have streamlit downloaded by doing 'pip install streamlit'
4. Please install pinecone 'pip install pinecone'
5. Please install sentence transformers 'pip install sentence-transformers'
6. Run 'python -m streamlit run TravelMinds.py'
